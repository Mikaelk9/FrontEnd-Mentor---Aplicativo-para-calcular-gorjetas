## Visão Geral

### O Desafio

Os usuários devem ser capazes de:

- Visualizar o layout ideal do app dependendo do tamanho da tela do dispositivo
- Ver os estados de hover para todos os elementos interativos da página
- Calcular corretamente a gorjeta e o custo total da conta por pessoa

## Meu Processo

### Construído com

- HTML5 – Estrutura semântica e organização do conteúdo.
- CSS3 – Estilização, design responsivo e layout em grid.
- JavaScript (Vanilla JS) – Manipulação dinâmica dos dados e interatividade dos botões.
- Google Fonts (Space Mono) – Fonte personalizada para a tipografia do projeto.
- Frontend Mentor – Desafio base utilizado como referência para o desenvolvimento.
